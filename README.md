# JavaSystem

JavaSystem is an experimental web system implemented in Java to explore how web infrastructure works at a low level.

The goal of the project is to understand the internal mechanics of web servers and distributed system architecture by building the components manually instead of relying on existing frameworks.

The system currently includes a minimal backend server and a basic frontend interface.

---

# Current Status

The project is in an early development stage.

At the moment the system includes:

* a backend server written in Java
* a simple frontend page
* a modular directory structure prepared for services, caching and database layers

The architecture is designed to evolve into a distributed system composed of multiple components.

---

# System Architecture

The project is structured around several layers.

```
Client (Browser)
      │
      ▼
Frontend (HTML)
      │
      ▼
Backend Server
      │
      ▼
Services Layer
      │
      ├── Cache
      ├── Database
      └── Logs
```

The backend server will act as the central component responsible for handling client requests and coordinating internal services.

# Project Structure

```
backend/     backend server implementation
frontend/    frontend web interface
services/    service layer (future modules)
cache/       caching layer
database/    database related components
logs/        runtime logs
```

---

# Directory Details

### backend

Contains the backend server implementation.

```
backend/
   MainServer.java
```

This component is responsible for starting the server and managing incoming connections.

---

### frontend

Contains the web interface served to the client.

```
frontend/
   index.html
```

---

### services

Reserved for backend services that will handle system logic.

---

### cache

Directory intended for caching mechanisms that may be implemented later.

---

### database

Directory reserved for database-related modules.

---

### logs

Stores runtime logs generated by the system.

---

# Running the Server

Compile the backend server:

```
javac backend/MainServer.java
```

Run the server:

```
java backend.MainServer
```
